# Brianza Taxi Service

**Location:** `02_Tickets_Transportation/BrianzaTaxiService`

## Overview

Brianza Taxi Service is an advanced taxi system featuring user account management, VIP discounts, and comprehensive service layer architecture. The application manages user profiles, tracks ride history, applies discounts for premium members, calculates fares with service improvements, and generates detailed receipts. It demonstrates enterprise-level patterns with separation of concerns through multiple service classes.

## Key Features

- User registration with personal information (name, surname, contact, address)
- VIP customer management with discount application
- Multiple ride tracking and history
- Dynamic pricing with VIP discount support
- Comprehensive ticket model with validation
- Service-oriented architecture (InputService, PriceService, UserService, TemplateService)
- HTML receipt generation with customer information
- User statistics and ride summaries

## Technologies Used

- Model-View-Service architectural pattern
- Service Layer Pattern (separation of concerns)
- Object-Oriented Design (classes, encapsulation, validation)
- User account management
- Financial calculations with discount logic
- Template-based HTML generation
- File I/O operations
- Input validation and error handling

## Learning Objectives

Students learn professional application architecture with service layers, implement multi-class systems with clear responsibility delegation, manage user state across multiple operations, apply discount logic and conditional pricing, validate user input comprehensively, and generate complex formatted output based on user type. The project showcases enterprise patterns useful for scalable applications.
