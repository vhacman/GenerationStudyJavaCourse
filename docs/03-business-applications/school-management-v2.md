# School Management v2

**Location:** `03_Business_Applications/SchoolManagement_v2`

## Overview

School Management v2 is an advanced version extending the basic school management system with MVC architecture, service layers, and file I/O capabilities. The application adds view components, file persistence, and template-based rendering to the core entity model from v1, demonstrating a complete business application architecture.

## Key Features

- Enhanced Person, Student, and Teacher entity models
- View layer for displaying entity information
- File I/O for persisting school data
- Template-based rendering of entity information
- Service layer for business logic separation
- Console input/output management
- HTML export of school records
- Demo classes for testing entity relationships
- Multi-entity management and display

## Technologies Used

- MVC architectural pattern
- Inheritance and polymorphism
- View layer implementation
- Template-based rendering
- File I/O and persistence
- Service layer design
- LocalDate for date handling
- Console-based user interaction
- Entity modeling and relationships

## Learning Objectives

This project helps to learn to build complete school management systems using MVC architecture, separate business logic with service layers, implement view layers for data display, persist data to files, and generate formatted reports from entity data.
