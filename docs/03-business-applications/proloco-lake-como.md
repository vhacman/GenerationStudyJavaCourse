# ProLoco Lake Como

**Location:** `03_Business_Applications/ProlocoLakeComo`

## Overview

ProLoco Lake Como is a tour guide recruitment and evaluation system for the Lake Como tourism promotion organization. The application evaluates candidates for tour guide positions based on age, location, education, and subject expertise, calculates qualification scores, and archives applications. It demonstrates scoring systems for professional recruitment.

## Key Features

- Evaluate candidate age eligibility (18-50 years)
- Apply geographic proximity scoring
- Assess education level qualifications
- Evaluate subject expertise in tourism
- Calculate combined qualification score
- Archive applications with unique numbers
- Provincial residency bonus points
- Educational attainment weighting
- Tourism subject specialization recognition

## Technologies Used

- Age validation and eligibility checking
- Geographic scoring based on provinces
- Qualification point systems
- LocalDate for current date handling
- File I/O for application archiving
- String-based data storage
- Scoring algorithm implementation

## Learning Objectives

Students learn to implement recruitment evaluation systems with weighted scoring, apply geographic-based qualifications, create professional application archiving, and build tour guide selection systems for tourism organizations.
