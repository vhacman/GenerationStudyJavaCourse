# Repair Shop

**Location:** `03_Business_Applications/RepairShop`

## Overview

Repair Shop is a business management system for tracking repair orders and calculating service costs. The application records repair jobs, calculates labor costs based on hours worked, material costs, and generates HTML invoices for customers. It demonstrates service-based business operations with cost calculation and documentation.

## Key Features

- Create and track repair orders with unique IDs
- Capture customer contact information and repair details
- Record material costs and hours worked
- Calculate total repair costs with labor charges
- Generate professional repair invoices in HTML format
- Store and view repair history
- Validate repair data before processing
- Test repair cost calculations with JUnit

## Technologies Used

- Object-oriented Repair entity design
- Cost calculation algorithms
- Unit testing with JUnit framework
- Template-based HTML invoice generation
- File I/O operations
- Data validation patterns
- Console-based user interface

## Learning Objectives

This project helps to learn to model a service business with cost calculations, implement invoice generation systems, validate business data, test financial calculations thoroughly, and manage customer information professionally.
