# Vacanze Romane

**Location:** `03_Business_Applications/VacanzeRomane`

## Overview

Vacanze Romane is a vacation package management system for weekend trips in Rome. The application combines hotel bookings, tour guide services, and attraction visits into comprehensive vacation packages. It demonstrates advanced system design with multiple entity types, service orchestration, and statistical reporting.

## Key Features

- Create vacation packages combining multiple services
- Manage hotel accommodations with pricing
- Organize tour guide assignments and scheduling
- Include tourist attractions with descriptions
- Calculate total vacation costs
- Generate vacation itineraries and confirmations
- Track vacation statistics and bookings
- Support data persistence and loading
- HTML export of vacation packages
- Wizard-based package creation interface

## Technologies Used

- MVC architectural pattern
- Repository pattern for data persistence
- Service layer for business logic
- Multiple entity types (Hotel, TourGuide, Attraction, WeekendTrip)
- Entity relationships and composition
- File I/O and data serialization
- Template-based HTML generation
- Statistical data aggregation
- Console wizard interface

## Learning Objectives

This project helps to learn to design complex vacation management systems with multiple related entities, implement service orchestration across different business domains, manage multi-part package pricing, generate comprehensive vacation confirmations, and build complete travel booking applications.
