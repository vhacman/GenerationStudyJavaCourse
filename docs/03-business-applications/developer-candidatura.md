# Developer Candidatura

**Location:** `03_Business_Applications/DeveloperCandidatura`

## Overview

Developer Candidatura is a job application system for developer positions. The application evaluates candidate qualifications including education, experience, technical skills, and travel availability, calculates a scoring system, and generates application documents. It demonstrates business logic for candidate evaluation and qualification assessment.

## Key Features

- Evaluate candidate age and geographic eligibility
- Assess educational background and qualifications
- Calculate experience level points
- Evaluate technical skills (Java knowledge)
- Consider travel availability as a qualification factor
- Calculate total candidacy score
- Generate application documents with scores
- Save applications with unique candidate numbers
- Create HTML application confirmations

## Technologies Used

- Candidate scoring algorithms
- Geographic validation and weighting
- Experience-based point systems
- Template-based document generation
- File I/O with archive system
- Input validation with multiple criteria
- HTML document generation

## Learning Objectives

Students learn to build candidate evaluation systems with weighted scoring, implement geographic eligibility checks, create professional application documentation, validate multiple qualification criteria, and generate evaluation reports.
