# Villa Melzi

**Location:** `03_Business_Applications/VillaMelzi`

## Overview

Villa Melzi is a ticket purchasing system for a botanical garden attraction. The application manages visitor admissions with complex discount rules based on residency, age, and student status, calculates ticket prices with cascading discounts, and records visitor statistics. It demonstrates multi-factor pricing with non-cumulative and cumulative discount rules.

## Key Features

- Collect visitor information and demographics
- Apply geographic-based pricing (local residency discounts)
- Implement age-based discounts (senior and child rates)
- Manage university student discount eligibility
- Handle complex discount rules (cumulative vs non-cumulative)
- Calculate final ticket prices with discounts
- Record visitor statistics for analysis
- Generate attendance data files
- Support multiple discount criteria

## Technologies Used

- Complex conditional pricing logic
- Case-insensitive string matching
- Age-based discount calculations
- Geographic location processing
- File I/O for statistics archiving
- Multi-factor discount algorithms
- Visitor demographic data collection

## Learning Objectives

Students learn to implement complex discount systems with business rules, handle non-cumulative geographic discounts vs cumulative social discounts, collect and analyze visitor statistics, and build attraction ticketing systems with sophisticated pricing logic.
