# Java Bank

**Location:** `03_Business_Applications/JavaBank`

## Overview

Java Bank is a banking system that manages client accounts and transactions. The application allows users to create bank accounts for clients, perform deposits and withdrawals, and view account balances. It demonstrates fundamental banking operations with user input validation and file-based reporting in HTML format.

## Key Features

- Create and manage bank accounts with client information
- Perform deposit transactions with balance updates
- Execute withdrawal operations with validation
- Display account balance and client details
- Generate HTML reports for account statements
- Persistent client storage and management
- Account ID-based organization

## Technologies Used

- Object-Oriented Programming (Encapsulation, Classes)
- Service Layer Pattern (AccountService, UserInputService)
- MVC Architecture (View, Model, Service layers)
- BigDecimal/numerical operations for financial calculations
- File I/O for HTML report generation
- Template-based rendering

## Learning Objectives

This project helps to learn how to structure a multi-layer banking application using the MVC pattern, implement business logic separation with service classes, handle financial transactions safely, validate user input for banking operations, and generate professional HTML reports from application data.
