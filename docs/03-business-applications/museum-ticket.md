# Museum Ticket

**Location:** `03_Business_Applications/MuseumTicket`

## Overview

Museum Ticket is a ticketing system for museum admission with age-based and income-based pricing. The application calculates ticket prices considering visitor age, annual income, and selected visit class, then generates HTML tickets for admission. It demonstrates business logic with multiple pricing factors and progressive pricing strategies.

## Key Features

- Calculate museum admission prices based on age and income
- Support multiple visit class options
- Implement income-based discount algorithms
- Age-based pricing adjustments
- Generate HTML admission tickets
- Capture visitor demographic information
- Dynamic pricing file generation
- Professional ticket formatting

## Technologies Used

- Income-based pricing calculations
- Age-based discount logic
- Template-based HTML generation
- File I/O operations
- String replacement for template processing
- Visitor data collection and processing

## Learning Objectives

Students learn to implement progressive pricing systems with multiple factors, calculate discounts based on demographic data, generate official admission documents, and build ticketing systems for cultural institutions.
