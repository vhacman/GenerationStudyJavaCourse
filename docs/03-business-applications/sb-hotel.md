# SB Hotel

**Location:** `03_Business_Applications/SBHotel`

## Overview

SB Hotel is a comprehensive hotel management system for booking rooms, managing reservations, and processing cancellations. The application handles multi-guest bookings, calculates room rates by category, offers shuttle services, processes refunds with policies, and generates HTML confirmation documents. It demonstrates complex business logic for hospitality management.

## Key Features

- Book rooms with multi-guest support (1-4 guests)
- Support multiple room types with different nightly rates
- Add shuttle service as optional additional cost
- Calculate stay duration and total costs
- Process booking cancellations with refund policies
- Generate HTML booking confirmations
- Create refund vouchers for cancellations
- Unique booking code generation
- Guest list management

## Technologies Used

- Complex business logic with multiple pricing rules
- LocalDate for date-based calculations
- Template-based HTML generation
- File I/O with organized file structure
- Shuttle service calculations
- Refund policy implementation
- Multi-step data collection and validation

## Learning Objectives

This project helps to learn to build hotel management systems with complex pricing logic, implement refund policies with business constraints, calculate multi-component costs, generate professional HTML documents, and handle multi-guest reservations with validation.
