# Lecco B&B

**Location:** `03_Business_Applications/LeccoB&B`

## Overview

Lecco B&B is a bed-and-breakfast booking system that manages room reservations with dynamic pricing. The application handles guest information, calculates room costs based on type and duration, applies membership discounts, and generates HTML booking confirmations. It demonstrates real-world hospitality business logic implementation.

## Key Features

- Create and manage room bookings with guest information
- Support multiple room types (BASIC, MIDDLE, SUPERIOR, SUITE)
- Calculate stay duration and nightly costs
- Apply membership discount tiers (NONE, SILVER, GOLD)
- Calculate final price with discount application
- Generate HTML booking confirmation documents
- Validate booking data and prevent invalid reservations

## Technologies Used

- Enumeration types (RoomType, MembershipType)
- LocalDate for check-in/check-out date handling
- Discount and pricing calculation logic
- Template-based HTML generation
- File I/O operations
- Data validation patterns
- Guest age calculation from date of birth

## Learning Objectives

This project helps to learn to implement hospitality business logic with multi-criteria pricing, manage date ranges for duration calculation, work with enumeration types for business rules, apply progressive discount systems, and generate professional booking confirmations.
