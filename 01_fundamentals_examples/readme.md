# Fondamenti Java

Moduli didattici per l'apprendimento dei concetti base della programmazione Java, dalla sintassi alla OOP avanzata.

</div>

---

## Percorso Consigliato

### Base
1. **SequenzaESelezione** — `if/else`, `switch`
2. **While** — Cicli base
3. **Modulo4** — Cicli `for`, `for-each`

### OOP Fondamentale
4. **Modulo 6** — Classi e oggetti
5. **Modulo7** — Incapsulamento, visibilita
6. **Modulo8** — Ereditarieta, `extends`, `super`
7. **Modulo9** — Classi astratte

### OOP Avanzato
8. **Modulo10** — Interfacce, polimorfismo
9. **Modulo11** — Lambda, Stream API, interfacce funzionali

### Pratica
10. **AlgoritmiDemo** — Strutture dati e Collections
11. **Examples** — Esempi pratici
12. **Recap** — Ripasso e consolidamento

---

## Contenuti

| Directory | Argomento | Temi chiave |
|-----------|-----------|-------------|
| `SequenzaESelezione/` | Istruzioni condizionali | `if/else`, `switch`, operatore ternario |
| `While/` | Iterazione con while | `while`, `do-while`, condizioni di uscita |
| `Modulo4/` | Cicli e iterazioni | `for`, `for-each`, `break`, `continue` |
| `Modulo 6/` | Classi e oggetti | `class`, costruttori, `new`, riferimenti |
| `Modulo7Incapsulamento/` | Incapsulamento | `private`, `public`, getter/setter |
| `Modulo8Ereditarietà/` | Ereditarietà | `extends`, `super`, `@Override` |
| `Modulo9Astrazione/` | Classi astratte | `abstract`, contratti, template |
| `Modulo10/` | Interfacce e polimorfismo | `implements`, binding dinamico |
| `Modulo11/` | Lambda e funzionali | Lambda, `removeIf`, Strategy Pattern |
| `AlgoritmiDemo - Strutture Dati/` | Collections | Array, ArrayList, HashSet, HashMap |
| `Examples/` | Sintassi base | Variabili, operatori, stringhe |
| `Recap/` | Ripasso | Consolidamento concetti |
