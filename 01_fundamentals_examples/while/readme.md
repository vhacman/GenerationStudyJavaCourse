---

## Concetti Chiave

### while

```java
while (condizione) {
    // eseguito finché la condizione è vera
}
```

### do-while

```java
do {
    // eseguito almeno una volta
} while (condizione);
```

### Differenza fondamentale

| Ciclo | Valutazione | Esecuzione minima |
|-------|-------------|-------------------|
| `while` | Prima di eseguire | 0 volte |
| `do-while` | Dopo aver eseguito | 1 volta |
