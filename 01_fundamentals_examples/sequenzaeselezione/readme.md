# Sequenza e Selezione

Istruzioni condizionali in Java: `if/else`, `switch` e operatore ternario.

</div>

---

## Argomenti

- Istruzione `if` semplice
- `if-else` e `if-else if` annidati
- Operatore ternario `? :`
- Istruzione `switch`
- Switch expression (Java 14+)

---

## Concetti Chiave

### if-else

```java
if (condizione) {
    // blocco vero
} else if (condizione2) {
    // blocco alternativo
} else {
    // blocco default
}
```

### switch

```java
switch (variabile) {
    case valore1:
        // codice
        break;
    case valore2:
        // codice
        break;
    default:
        // codice
}
```
