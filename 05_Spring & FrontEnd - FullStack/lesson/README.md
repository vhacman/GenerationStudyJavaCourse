# Lesson - Progetto Lezioni

Progetto Java per la gestione di lezioni.

## Struttura

Directory contenente file compilati e output di build per il progetto lesson.

## Nota

Questo progetto contiene principalmente file compilati (.class) e risorse di build.
I sorgenti potrebbero essere stati spostati o integrati in altri progetti come GenSchool.

## Build

```bash
# Se presente un file pom.xml o build.gradle
mvn clean install
# oppure
gradle build
```
